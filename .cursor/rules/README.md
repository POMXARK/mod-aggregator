# Cursor AI Rules

Эта директория содержит правила для Cursor AI, организованные по категориям.

## Структура

- `frontend.mdc` - правила для работы с Svelte 5 frontend
- `backend.mdc` - правила для работы с Rust backend (Tauri)
- `code-patterns.mdc` - общие паттерны кода и лучшие практики

## Формат файлов

Файлы используют формат `.mdc` (Markdown с YAML frontmatter):

```yaml
---
description: Описание когда применяется правило
globs: src/**/*.svelte  # Паттерны файлов для автоматического применения
---
```

## Использование

Cursor AI автоматически применяет правила из этой директории в зависимости от:
- Открытых файлов (по glob паттернам)
- Контекста задачи
- Типа файлов, с которыми работаешь

## Добавление новых правил

1. Создай новый `.mdc` файл в этой директории
2. Добавь YAML frontmatter с описанием и glob паттернами
3. Напиши правила в Markdown формате
4. Правила будут автоматически применяться Cursor AI
