# Инструкция по настройке

## Быстрый старт

1. Установите зависимости:
```bash
npm install
```

2. Запустите в режиме разработки:
```bash
npm run tauri:dev
```

## Добавление парсера для synthira.ru

1. Откройте приложение
2. Перейдите в раздел "Сайты"
3. Нажмите "Добавить сайт"
4. Заполните форму:
   - **Название**: Synthira.ru
   - **URL**: https://synthira.ru
   - **URL списка модов**: https://synthira.ru/sims4 (или нужный раздел)
   - **Селектор списка модов**: `.mod-item` или `.article-item` (зависит от структуры сайта)
   - **Селектор названия**: `h2 a` или `.title a`
   - **Селектор ссылки**: `h2 a` или `.title a`
   - **Селектор версии**: `.version` (если есть)
   - **Селектор автора**: `.author` (если есть)
   - **Селектор изображения**: `img` или `.thumbnail img`

5. Используйте конструктор парсеров для визуального создания и тестирования селекторов

## Тестирование селекторов

1. Перейдите в раздел "Конструктор"
2. Введите URL страницы для теста
3. Введите CSS селектор
4. Нажмите "Тестировать"
5. Просмотрите результаты и скорректируйте селекторы при необходимости

## Автоматическая проверка обновлений

Приложение автоматически проверяет обновления каждый час. Вы также можете запустить проверку вручную:
- Перейдите в раздел "Моды"
- Нажмите "Проверить обновления"

## Уведомления

При обнаружении обновлений модов вы получите системное уведомление. Все уведомления сохраняются в разделе "Уведомления".
