# Библиотеки и утилиты

Утилиты и переиспользуемая логика для frontend.

## Файлы

### tauri-wrapper.ts
Обертка для вызова Tauri команд с обработкой ошибок.

### tauri-mock.ts
Mock реализация Tauri API для работы в браузере (режим разработки).

### api.ts
TypeScript интерфейсы для данных приложения:
- `Site` - Сайт
- `Mod` - Мод
- `Notification` - Уведомление

## Composables

Переиспользуемая логика, отделенная от компонентов:

### useSites.ts
Работа с сайтами:
- `loadSites()` - Загрузить список сайтов
- `deleteSite()` - Удалить сайт
- `saveSite()` - Сохранить сайт

### useMods.ts
Работа с модами:
- `loadMods()` - Загрузить список модов
- `checkUpdates()` - Проверить обновления

### useNotifications.ts
Работа с уведомлениями:
- `loadNotifications()` - Загрузить уведомления
- `markAsRead()` - Отметить прочитанным
- `markAllAsRead()` - Отметить все прочитанными

### useUI.ts
Управление UI состоянием:
- `setPage()` - Установить текущую страницу
- `toggleSidebar()` - Переключить сайдбар
- `setSelectedSite()` - Установить выбранный сайт
